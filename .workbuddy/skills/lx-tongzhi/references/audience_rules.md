# lx-tongzhi 人群视角规则

## 商家 / 运营主体（shangjia）

优先回答：

- 这件事和哪个运营主体、城市、品牌有关。
- 商家需要做什么。
- 截止时间是什么。
- 需要填写、检查或反馈哪个链接。
- 不做会影响什么。

避免：

- 司机端过细权益解释。
- 内部判断过程。
- 后台字段名、内部补贴属性、策略标签。

## 司机（siji）

优先回答：

- 有什么权益。
- 哪天、哪个时段生效。
- 如何参与。
- 是否需要购买卡券、完成订单或满足新人条件。
- 二维码/海报/链接是否可直接扫码查看。

避免：

- 商家提报动作。
- 后台配置步骤。
- 内部补贴属性。

## 线下渠道（qudao）

优先回答：

- 对外怎么说。
- 需要催谁、催什么、什么时候前反馈。
- 遇到异常怎么收集信息。
- 哪些口径不能说错。

避免：

- 给司机或商家的长篇原文照搬。
- 未确认的承诺。
- 内部敏感词。
