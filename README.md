## 设计原则

- **编号前缀**：00-12 保证目录按业务逻辑排序，10 用于表格同步，12 用于农夫协作文档
- **输入输出分离**：Pipeline 场景统一为待处理→已处理+处理日志
- **原表保留**：原始文件进存档，永不被覆盖
- **日志可追溯**：每个 Pipeline 场景自带处理日志
- **码表在线化**：共享模板不分发本地码表 Excel，统一通过 `lx_shujuku` 查询公司库 `operator_brand`
